## THE VISION

This is a **Cognitive Translation Layer** — an immersive, evolving, self-recursive experience for neurodivergent minds.

The system helps users understand:
- How they think (not how they should)
- How they regulate (not how to "fix" themselves)
- What environments support them
- Recurring patterns in their cognition
- Cognitive strengths (not just challenges)
- Emotional states as signals, not defects

**Evolution path:**
Assessment → Cognitive Map → Adaptive Toolkit → Pattern Detection → Cognitive Companion → Personal Operating System

**Design Principles:**
1. Meaning before productivity
2. Understanding before optimization
3. Strengths and challenges equally visible
4. Emotional states are signals, not defects
5. Recommendations emerge from patterns
6. System learns continuously from user interaction

**Audience:**
- Primary: Neurodivergent individuals
- Expanded: Non-linear thinkers (creatives, founders, researchers, gifted, systems thinkers)

**AVOID:**
- Generic ADHD advice
- Habit-tracker mentality
- Productivity-first framing
- Excessive gamification
- Negative framing ("this is not...")
- Word "regulate" (pejorative)
- Word "mythic" (use "arcane" instead)

## TWO ENTRY POINTS

### Entry 1: "Am I 2E?" — SPARK Assessment
- For gifted/twice-exceptional people
- Dabrowski's Theory of Positive Disintegration
- 5 dimensions: Cognitive Intensity, Emotional Resonance, Sensory Amplification, Creative Divergence, Existential Drive
- 40 questions, Likert 0-4
- Scoring: 0-20 total
- Profile types: Quiet Spark (0-6), Flickering Spark (7-12), Burning Spark (13-16), Wildfire (17-20)
- Results: animated radar chart, profile reveal, personalized interpretation

### Entry 2: "Daily Cognitive OS" — Onboarding → Dashboard
- For all neurodivergent people
- Emotional Navigator, Sensory Check-In, Discoveries, Pattern Engine, Recommendations
- Tracks patterns over time
- Self-recursive: system evolves with user
